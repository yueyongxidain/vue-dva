# vuetest1

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
目录结构：
 assets //静态资源
 components   //各个组件
 modules   //各个模块式的model
 store     
    createUrlStore   //用于合成store 和路由的   所有的路由和菜单都写在routerConfig 数字的跟菜单下面  
 views    //写一些单页面整体布局文件的
 main.js   //主要修改element ui库按需加载
